2)WAIT IN MAIN MENU TILL ACTION RECIEVED

3)SET TARGET PARAMETERS FOR TEMPERATURE,HUMIDITY,ROTATION,TIME

4)READ VALUES FROM DHT22 SENSOR

5)COMPARE CURRENT AND TARGET VALUES

10)IF RESET IS LOW CONTINUE
